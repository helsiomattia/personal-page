Quero alterar o deploy do meu projeto React + Vite no GitHub Pages.

Hoje o projeto usa gh-pages e cria uma branch gh-pages para publicar o site, mas eu quero parar de usar essa branch.

Objetivo:
- Manter somente a branch main como branch principal de trabalho.
- Sempre que eu fizer push na main, o GitHub deve rodar npm install, npm run build e publicar automaticamente no GitHub Pages.
- Não quero mais precisar rodar npm run deploy.
- Não quero mais depender da branch gh-pages.

Projeto:
- React + Vite
- Repositório: https://github.com/helsiomattia/personal-page
- URL final: https://helsiomattia.github.io/personal-page/
- A configuração do Vite deve continuar com base: '/personal-page/'

Faça as seguintes alterações:

1. Verifique o arquivo vite.config.js

Ele deve conter:

import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
  plugins: [react()],
  base: '/personal-page/',
});

Não remova o base.

2. Remova o uso de gh-pages do package.json

No package.json, remova o script:

"deploy": "gh-pages -d dist"

Se existir dependência gh-pages em devDependencies, remova também.

Os scripts devem ficar parecidos com:

"scripts": {
  "dev": "vite",
  "build": "vite build",
  "preview": "vite preview"
}

Preserve scripts úteis adicionais se existirem.

3. Criar workflow de deploy com GitHub Actions

Crie a pasta:

.github/workflows

Dentro dela, crie o arquivo:

deploy.yml

Com este conteúdo:

name: Deploy Vite site to GitHub Pages

on:
  push:
    branches:
      - main
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

concurrency:
  group: github-pages
  cancel-in-progress: true

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout repository
        uses: actions/checkout@v4

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm

      - name: Install dependencies
        run: npm ci

      - name: Build project
        run: npm run build

      - name: Upload GitHub Pages artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: dist

  deploy:
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}

    runs-on: ubuntu-latest
    needs: build

    steps:
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4

4. Validar localmente

Depois das alterações, rode:

npm install
npm run build

O build precisa passar sem erro.

5. Me entregue o resumo final

Ao final, me diga:
- Quais arquivos foram alterados.
- Se o gh-pages foi removido do package.json.
- Se o workflow .github/workflows/deploy.yml foi criado.
- Se o npm run build passou.
- Quais comandos eu devo rodar para subir.

Não execute deploy manual.
Não use mais npm run deploy.
